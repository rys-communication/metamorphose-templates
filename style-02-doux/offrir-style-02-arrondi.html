Cartes Cadeaux
Comment fonctionne une carte cadeau ?
Vous choisissez la carte, payez en ligne, recevez un bon numérique par email immédiatement. La bénéficiaire réserve sur la plateforme avec ce code. Valable 1 an.
Comment la carte cadeau est-elle transmise ?
Le bon est envoyé par email dès la validation. Vous pouvez l'imprimer ou le transmettre par message. Pas d'envoi postal.
Combien de temps est valable une carte cadeau ?
Les cartes cadeaux sont valables 1 an à compter de la date d'achat.
Peut-on offrir plusieurs cartes cadeaux différentes ?
Oui. Chaque carte est indépendante et peut être transmise à la même personne pour composer un cadeau sur mesure.
Packs
Comment fonctionne un pack ?
Après l'achat, vous recevez un code unique. Vous l'utilisez lors de chaque réservation pour les prestations incluses, dans l'ordre et au rythme qui vous convient.
Un pack peut-il être offert ?
Absolument. Tous les packs sont offrables. Le code reçu par email se transmet facilement.
Comment fonctionne le paiement en 2 fois ?
Disponible sur tous les packs via la plateforme de réservation. Deux prélèvements distincts, sans frais supplémentaires.
Peut-on choisir l'ordre des séances ?
Oui. Vous réservez chaque prestation quand vous le souhaitez. Le code reste valable jusqu'à utilisation complète.
